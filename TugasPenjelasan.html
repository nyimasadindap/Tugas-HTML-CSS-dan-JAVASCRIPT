HTML, CSS dan JavaScript

Nyimas Adinda Paradiza | 202331041

1. HTML

HTML adalah Singkatan dari Hypertext Markup Language yang digunakan untuk membuat struktur halaman website yang menarik dan dapat diakses melalui internet.

Mengapa kita menggunakan HTML:

  • a. Membuat Kerangka Halaman Web
  • b. Bekerja Sama dengan CSS dan JavaScript
  • c. Mudah Dipakai dan Dipelajari HTML itu sederhana, bahkan pemula pun bisa langsung mencobanya.

Bagaimana HTML diintegrasikan:

HTML bisa diintegrasikan dengan teknologi lain untuk membuat sebuah halaman web atau aplikasi lebih menarik, interaktif, dan fungsional. Integrasi ini melibatkan CSS, JavaScript, dan berbagai tools atau layanan lainnya. Berikut cara-cara utama mengintegrasikan HTML:

  • a. Integrasi dengan CSS (Desain dan Tata Letak)
  • b. Integrasi dengan JavaScript (Interaktivitas)
  • c. Integrasi dengan Framework dan Library

2. CSS

Cascading Style Sheets (CSS) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak halaman web. Jika HTML memberikan struktur (kerangka) halaman, CSS adalah "cat dan dekorasi" yang membuat halaman web terlihat menarik.

Mengapa kita menggunakan CSS:

  • a. Mengatur Tampilan Halaman Web
  • b. Memisahkan Konten dan Desain
  • c. Konsistensi di Seluruh Halaman

Bagaimana CSS diintegrasikan:

CSS dapat diintegrasikan ke HTML dengan tiga cara utama, tergantung pada kebutuhan dan skala proyek. Berikut adalah cara integrasi CSS:

  • a. Inline CSS → Untuk styling cepat atau elemen spesifik.
  • b. Internal CSS → Untuk styling halaman tunggal.
  • c. Eksternal CSS → Untuk desain terpusat dan konsisten di banyak halaman. Metode ini sering dikombinasikan untuk fleksibilitas dalam pengembangan web.

3. JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif dan dinamis. JavaScript memungkinkan pengembang untuk menambahkan fungsionalitas ke dalam halaman web seperti animasi, interaksi pengguna, dan pengolahan data secara langsung di browser.

Mengapa kita menggunakan JavaScript:

  • a. Menambahkan Interaktivitas pada Halaman Web
  • b. Memanipulasi Halaman Web (DOM Manipulation)
  • c. Validasi Formulir

Bagaimana JavaScript diintegrasikan:

JavaScript diintegrasikan dengan HTML dan CSS untuk membuat halaman web yang dinamis dan interaktif. Berikut cara-cara umum mengintegrasikan JavaScript ke dalam proyek web:

  • a. Inline: Untuk aksi langsung di elemen HTML.
  • b. Internal: Dalam tag <script> di file HTML untuk halaman tunggal.
  • c. Eksternal: Dengan file .js terpisah untuk proyek besar.
  • d. Dengan CSS: Untuk perubahan gaya dinamis.
  • e. Dengan API: Untuk mengambil data dari server tanpa memuat ulang halaman.
